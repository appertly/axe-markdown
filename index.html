The Gist

AXE Markdown takes Markdown text and converts it into bona fide XHP nodes.

Features

The axe:markdown tag takes Markdown text and renders XHP. For example, take Markdown like this:

# Hello!
This is great
* Don't you think?

…give it to <axe:markdown text={$text}/> and get XHP nodes like exactly this:

<x:frag>
    <h1>Hello!</h1>
    <p>This is great</p>
    <ul>
        <li>Don't you think?</li>
    </ul>
</x:frag>

Under the Covers

Behind the scenes, we use the league/commonmark library to generate the Markdown AST. That being the case, you can supply your own customized Environment or DocParser.

$environment = League\CommonMark\Environment::createCommonMarkEnvironment();
// customize Environment
$docParser = new League\CommonMark\DocParser($environment);
// customize DocParser
echo <axe:markdown text={$text} docParser={$docParser}/>;
